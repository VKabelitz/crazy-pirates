# GameJam – Dieses andere Adventure

Dieses Projekt entstand im Rahmen eines dreitägigen Game Jams. Das Ergebnis ist ein Tower‑Defense‑Spiel, in dem verschiedene Verteidigungstürme – darunter Kanonen, Magietürme und Armbrusttürme – strategisch platziert werden müssen, um die heranfliegenden **Helitoads** und **Eismonster** aufzuhalten.

Das gesamte Geschehen spielt auf einem schwebenden Piratenschiff. Die Gegner versuchen, in das Innere des Schiffs einzudringen – und deine Aufgabe ist es, sie daran zu hindern.

Unser Entwicklerteam trägt den Namen **Richards Haus**.
